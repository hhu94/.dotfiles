[dotfiles](https://github.com/holman/dotfiles). Logo from

## Installation

Clone this repo:
```
git clone https://github.com/hhu94/.dotfiles ~/.dotfiles
```
Bootstrap!:
```
~/.dotfiles/bootstrap
```

See [README](https://github.com/hhu94/.dotfiles/blob/master/vim/README.md) to
setup Vim.
