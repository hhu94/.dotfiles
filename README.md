These are my dotfiles.

## Installation

Clone this repo:
```
git clone https://github.com/hhu94/.dotfiles ~/.dotfiles
```
Bootstrap!:
```
~/.dotfiles/bootstrap
```

See [vim/README](https://github.com/hhu94/.dotfiles/blob/master/vim/README.md)
for instructions on how to setup Vim.
