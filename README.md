# ![dotfiles][logo]

These are my dotfiles. Includes a bin directory that symlinks its contents to
~/bin and my personal vim configuration.

## Installation

Clone and bootstrap:
```
git clone https://github.com/hhu94/.dotfiles ~/.dotfiles
~/.dotfiles/bootstrap
```
